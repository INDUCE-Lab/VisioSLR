## Source Code Structure

1. Python source for fine-tuning YOLOv8, v9, or v11, and implementation in a server-client environment for real-time sign language recognition.

2. Dataset that is used in this work.


## Camera Setup:

We use Intel RealSense D455 camera:

To set up the camera, guidelines are available on the following link: https://github.com/IntelRealSense/librealsense

To remove the pattern dots projected by the camera, adjust the "laser power" slider in the Controls section of the RealSense Viewer.
